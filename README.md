# 🍔 DevBurguer

Um mini sistema de pedidos inspirado em apps de delivery como iFood, desenvolvido com **HTML, CSS e JavaScript puro**.

---

## 📌 Sobre o projeto

O **DevBurguer** é uma aplicação web que simula um catálogo de produtos com carrinho de compras, permitindo ao usuário:

* visualizar produtos
* adicionar itens ao carrinho
* remover itens individualmente
* aplicar descontos
* buscar produtos pelo nome
* visualizar o total da compra

O objetivo do projeto foi praticar **manipulação de DOM**, **lógica de programação** e **organização de código sem frameworks**.

---

## 🧠 Principais aprendizados

Durante o desenvolvimento, foram trabalhados conceitos importantes como:

* Manipulação dinâmica do DOM
* Organização de código (separação de responsabilidades)
* Uso de métodos de array:

  * `map`
  * `filter`
  * `reduce`
  * `find`
* Event Delegation
* Controle de estado da aplicação sem frameworks
* Atualização dinâmica de interface (renderização)
